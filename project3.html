“오늘은 어떤 코인이 많이 올랐는지
메인 화면에서 바로 확인하고 싶어요.”

코인원은 비트코인 거래만 제공했던 2014년 이후 한번도 메인 페이지의 개선이 없었습니다.
비트코인만이 존재했을 때 설계된 UI였기 때문에 비트코인의 시황만 주요하게 제공할 뿐 여러 코인들을 한 눈에 확인하기 힘들었습니다.
그러다 유저 인터뷰 중 타 거래소는 코인 등락률을 메인에서 제공해 가격이 급등한 큰 코인은 어떤 이슈가 있는지 살펴보게 된다는 경험을 듣게 되었습니다.

problem
  • UI 확장성 부족

    코인이 많이 상장하면 상단의 배너 영역이 불필요하게 넓어짐

  • 비트코인의 시황만 제공

    비트코인을 제외한 코인의 변동률을 확인하려면 거래소에 접속 해서 불편

  • 노출된 개인정보

    아이디가 접속하자 마자 노출 되는 보안상의 문제

  • 긴 스크롤

    스크롤이 길어지면 유저는 거래소로 바로 이동할 뿐 아래 내용까지 읽어 보지 않음

stakeholders

메인은 여러 이해관계가 충돌하는 페이지기 때문에, 단순히 UX의 개선에서 끝나는 것이 아니라
여러 팀의 의견을 듣고 이를 고려하여 설계할 수 있어야 했습니다.

  • 비즈니스 : 코인원 크립토 인덱스 추가

    코스피나 코스닥처럼 유저가 전체 시장 상황을 반영하는 지표를 확인할 수 있도록 한다.

  • 마케팅 : 노출 배너 갯수 증가

    현재는 이벤트를 한개만 노출 할 수 있다. 동시에 여러 이벤트를 노출 시켜 마케팅 효과를 증가 시키고 싶음.

  • 모바일 : 앱 다운로드 안내

    모바일 앱 다운로드율이 저조하다. 모바일 앱을 다운 할 수 있도록 웹에서 안내가 필요하다.

goal

메인의 역할 재정의

메인이 해당 코인의 거래 페이지로
바로 이동할 수 있는 게이트의 역할을 하게 만든다.

모든 코인의 시황 확인

모든 코인들의 시황을 확인 할 수 있게 하여
상세 거래페이지를 클릭하도록 유도한다.

user flow

  • 종목 확인

    어떤 코인들이 상장 되어 있는지 확인한다.

  • 가격 확인

    해당 코인의 현재가가 얼마인지 확인한다.

  • 등락률 확인

    전일대비 얼마만큼의 가격 변동이 있었는지 확인한다.

  • 거래량 확인

    해당 코인의 거래량이 얼만큼인지 자금의 유동성을 확인한다.
